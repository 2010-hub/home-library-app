\## 📖 Описание



\*\*Домашняя библиотека\*\* — это веб-приложение для каталогизации личной коллекции книг. Создано для работы на \*\*Raspberry Pi\*\*, но может запускаться на любом устройстве с Node.js.



\### ✨ Возможности



\- 👑 \*\*Администратор\*\* — полный контроль (добавление, редактирование, удаление, настройки)

\- 📖 \*\*Читатель\*\* — только просмотр (вход без пароля)

\- 📸 Добавление книг с обложкой

\- 🔍 Автоматический поиск по ISBN (Google Books API)

\- 📊 3 режима просмотра: карточки, таблица, галерея

\- 🔎 Поиск и сортировка по любым полям

\- ⚙️ Настройки: смена пароля, тема оформления, экспорт/импорт данных

\- 🔄 Обновление прошивки из GitHub
