## Документация

### 🔐 Два режима доступа
| Режим | Возможности |
|-------|-------------|
| **Администратор** | Добавление, редактирование, удаление книг, настройки системы (вход с паролем) |
| **Читатель** | Только просмотр каталога без возможности редактирования (вход без пароля) |

### 📚 Управление книгами
-  Добавление книг с обложкой (загрузка изображений)
-  Редактирование и удаление книг
-  Автоматический поиск по ISBN (Google Books API) (функция находится в разработке)
-  Поля: название, автор, ISBN, жанр, серия, издательство, год издания, кол-во страниц, место хранения, ссылка на электронную версию

### 🖼️ 3 режима просмотра каталога
| Режим | Описание |
|-------|----------|
|  **Карточки** | Визуальный просмотр с обложками |
|  **Таблица** | Все данные в виде таблицы |
|  **Галерея** | Крупные обложки для быстрого поиска (видно только название и обложка) |

### 🔎 Поиск и сортировка
- Поиск по названию, автору, ISBN, жанру, серии, издательству, месту хранения
- Сортировка по любому полю (по возрастанию/убыванию)

### ⚙️ Настройки
- Смена пароля администратора
- Изменение названия библиотеки
- Выбор темы оформления (светлая/тёмная)
- Настройка количества книг на странице
- Экспорт данных (JSON/CSV)
- Импорт данных из бэкапа
- Обновление прошивки из GitHub

### 🔒 Безопасность
- Данные хранятся в JSON файлах (без внешней БД)
- Простая установка без дополнительных зависимостей
- Разделение кода и данных (при обновлении данные сохраняются)

## Установка (Linux)
У вас должны быть установлены [зависимости проекта](https://github.com/2010-hub/home-library-app#зависимости)

**1. Клонирование репозитория**

```git clone https://github.com/2010-hub/home-library-app.git```

**2. Переход в директорию library-app**

```cd library-app```

**3. Запуск сервера**

```node server.js```

**4. Открыть в браузере**

Всё готово! Осталось только открыть ```http://<IP-адрес-вашего-устройства>:4002``` и ввести **Логин:** admin **Пароль:** admin

**5. Обновите пароль**

Перейдите в настройки (нажав на шестерёнку в правом верхнем углу) и измените пароль.

## Зависимости
Эта программа требует установки **Node.js**. Если вы заметили, что он данное ПО можно запустить на версии ниже, или он не работает на какой-либо версии, то напишите в [поддержку](https://github.com/2010-hub/home-library-app/issues/new/choose)
**1. Установка Node.js**
   
```curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -```

```sudo apt install -y nodejs```

**2. Проверка установки**
   
```node --version  # Должно быть v18.x.x```

```npm --version   # Должно быть v9.x.x или выше```
